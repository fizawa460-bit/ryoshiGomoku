# 量子五目並べ

1台のスマートフォンを2人で交互に使って遊ぶ、量子風の五目並べです。

## ルール

- 15×15の盤で黒が先手です。
- 白番は `白90% → 白70% → 白50% → 白30% → 白10%` の駒を順に置き、5種類を使うと最初へ戻ります。
- 黒番も `黒90% → 黒70% → 黒50% → 黒30% → 黒10%` の駒を順に置き、5種類を使うと最初へ戻ります。
- 盤上の駒が縦・横・斜めに5つ連続すると、観測が解禁されます。
- 観測すると、盤上の全駒がそれぞれの確率に従って一時的に白か黒へ確定します。
- 片方だけ五目が成立すればその色の勝ち、両方同時なら引き分けです。
- 勝敗がつかなければ観測をやめ、全駒を未確定に戻して対局を続けます。
- 観測中は新しい駒を置けません。

## 量子もつれ

- 黒と白は、それぞれ対局中に1回だけ量子もつれを使えます。
- 自分の手番で「もつれ」を押し、盤上にある未接続の自分の駒を1個選びます。
- 続けて空いている交点を選ぶと、次の駒が置かれ、2駒が線で結ばれます。この配置で手番を消費します。
- もつれた2駒は観測時に同じ乱数を共有します。各駒の確率自体は変わりませんが、観測結果が強く連動します。
- すでにもつれている駒を、別の駒と重ねてもつれさせることはできません。
- 駒を選ぶ前後に「もつれ取消」を押すと、使用回数を消費せず通常の配置へ戻れます。

## 起動

静的ファイルだけで動作します。`index.html` をブラウザで開くか、任意の静的Webサーバーで配信してください。
